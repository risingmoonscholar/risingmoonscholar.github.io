Semantic HTML Reference

Semantic Tags

Tag Name Description
<header> Header a container for introductory content or a set of navigational links
<nav> Nav used for major blocks of navigation links, usable by screen readers for accessibility.
<section> Section a thematic grouping of content, typically with a heading.
<aside> Aside designates content indirectly related to the surrounding content
<figure> Figure specifies self-contained content, like illustrations, diagrams, & photos.
<footer> Footer typically contains authorship information, copyright information, contact information, or a sitemap.
